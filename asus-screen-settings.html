NAME

asus-screen-settings - Graphical configuration for Asus Screen Toggle

SYNOPSIS

asus-screen-settings

DESCRIPTION

A GTK3-based graphical interface used to configure hardware parameters (Keyboard Vendor ID, Product ID), display output names, and global service behavior.

Modifying system-wide settings requires administrative privileges (invokes pkexec).
