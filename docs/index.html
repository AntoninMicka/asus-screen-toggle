Asus Screen Toggle v1.0.1

Automatic display management for Asus Zenbook Duo laptops.

Seamlessly toggle your secondary screen based on external keyboard connection and lid status. Built for Linux (Debian/KDE).

Features

  • Smart Detection: Automatically enables the bottom screen when the external keyboard is detached.
  • Orientation Support: Full support for screen rotation via iio-sensor-proxy.
  • GTK3 Interface: User-friendly settings app to configure display IDs and hardware names.
  • Systemd Integration: Reliable background service with minimal resource footprint.
