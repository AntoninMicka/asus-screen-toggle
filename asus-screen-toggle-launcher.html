NAME

asus-screen-toggle-launcher - Launcher for the Asus screen management agent

SYNOPSIS

asus-screen-toggle-launcher

DESCRIPTION

asus-screen-toggle-launcher is a wrapper script that safely executes the user agent. It checks if another instance is already running and ensures the environment is correctly set up for D-Bus communication and the display server.

This script is typically invoked automatically upon user login (via a .desktop file in autostart), but can also be run manually to restart the service after a crash or configuration change.

FEATURES

1. Instance check (prevents duplicates). 2. Initialization of state files in ˜/.local/state. 3. Execution of asus-user-agent.
