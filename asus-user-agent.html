NAME

asus-user-agent - Management agent for Asus Zenbook Duo secondary display

SYNOPSIS

asus-user-agent

DESCRIPTION

asus-user-agent is a background resident program that monitors keyboard connection status and hardware sensors. It displays an icon in the notification area (system tray), allowing the user to toggle between automatic mode and forced display on/off states.

OPTIONS

The program does not accept any command-line arguments. Configuration is handled via the GUI utility.

SIGNALS

SIGUSR1

Forces an immediate sensor status check (used by system-level triggers).

SIGHUP

Reloads the configuration files.
